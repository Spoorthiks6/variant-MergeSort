Merge Sort Variants in C

Three merge‑based sorting algorithms: classic merge sort,two‑way merge sort, and k‑way merge sort (using a min‑heap).

Compilation & Execution

gcc mergeSort.c -o mergeSort && ./mergeSort
gcc twoWayMerge.c -o twoWayMerge && ./twoWayMerge
gcc K-wayMerge.c -o K-WayMerge && ./kWayMerge
